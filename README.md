# For Tekarto

Simple code for Tekarto by NMCTs student

## Getting Started

### Dependencies

* A working browser!

### Installing

* Click on the triangle next to lime "Code" button
* Click Download ZIP
* Extract the downloaded zip!

### Executing program

* Just run it inside the browser!
